Domain to add to unqualified addresses (Options)

Specifies a domain to be added to unqualified names.

An unqualified name is one that doesn't have an "@" sign followed by a domain name. If you address a message to someone without including their domain name, Eudora automatically adds this domain to that address. This setting can be used as a time-saving device when addressing large numbers of messages to users who reside in the same domain.

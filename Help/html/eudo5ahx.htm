Use a Signature

A signature is a few lines of text that are added automatically to the end of an outgoing message when it is sent. A signature can be whatever you want, but it is mostly used to give contact information (telephone, address, and so forth). You use only one signature at a time in a message, but you can create as many different signatures as you want.

You can choose to see or hide your signature while composing your message. See Composing Mail Options Window.

For more information, see Creating a Signature and Including a Signature in Messages.

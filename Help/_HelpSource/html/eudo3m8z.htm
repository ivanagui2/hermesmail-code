Putting Multiple Users on One PC

To have multiple users on one computer, create shortcuts to the HERMES executable for each user. The mail directories can be named whatever you like, and put anywhere you like, including on floppies or network drives. To tell HERMES which directory to use, specify that directory in the Target field of your shortcut. A new HERMES.ini file will be created once the user launches HERMES.

To have multiple users on one computer, do the following:

1.For each user, create a shortcut to the HERMES executable file (HERMES.exe).

2.Right-click on the new shortcut and select Properties.

3.Click the Shortcut tab.

4.In the Target field, add the path to the user’s HERMES.ini file.

5.To start HERMES, users simply double-click on their shortcut. A new HERMES.ini file will be created for each user located in the directory specified in the Target: field.

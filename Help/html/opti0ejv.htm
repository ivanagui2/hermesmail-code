Add 'Not Junk'ed senders to Address Book

If Put not junked senders in the address book is turned on under Junk Mail Options, this option adds addresses of "Not Junk'ed" senders into the address book entered here. Important – you must create an Address Book file with the same name.

If Mail isn’t junk if sender is in an address book is selected from the Junk Mail Options list, any future email from this sender is not considered junk mail.

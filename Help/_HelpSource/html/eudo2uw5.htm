Changing the HERMES Mode

You can choose one of three operating modes on how you wish to use HERMES. They are as follows:

Paid mode

This mode provides the full-featured HERMES desktop email client without advertising.

Sponsored mode

This mode provides the full-featured HERMES desktop email client at no charge, supported by sponsors' advertising. It includes all of the features that are in the Paid mode.

Light mode

This is an upgrade to the freeware version formerly known as HERMES Light. It includes fewer features than the other modes and a sponsor image or logo, but no advertising.

If you bought HERMES from a retail store or from a web site, HERMES installs in paid mode. However, if you would like to change the operating mode, for example to sponsored mode, you can do so right in the HERMES program. (Remember to request a refund from QUALCOMM. See the last page of this guide for details.)

To change the HERMES operating mode after you install HERMES, do the following:

1.From the Help menu, choose Payment & Registration. The Payment and Registration window appears.

2.In the Which HERMES is right for you? section:

§To use HERMES in sponsored mode, click Sponsored Mode (free, with ads). HERMES displays the ad window.

§To use HERMES in paid mode, click Paid Mode (costs money, no ads). HERMES displays the HERMES web site for you to purchase HERMES.

§To use HERMES in light mode, click Light Mode (free, fewer features). HERMES changes to light mode by shutting down certain features. The features that will be off are listed in the Light mode dialog box.

3.In the Keeping Current section:

§To customize the ads, click Customize the Ads You See. HERMES displays a web site where you can provide information about yourself to help in customizing which ads should be displayed to you.

§To find the most current version of HERMES, click Find the Latest Versions. The HERMES web site displays where you can retrieve the latest version of HERMES.

You can tell which mode you are in because that button is grayed out.

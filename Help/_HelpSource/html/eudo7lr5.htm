Uninstalling HERMES

You can uninstall HERMES by using the tools provided with your Windows 95/98 or Windows NT 4.0 (or later) operating system. To uninstall HERMES, do the following:

1.From the Start menu, choose Settings, then Control Panel.

2.Open the Control Panel, double-click Add/Remove Programs, select HERMES Email, and click Remove. HERMES will uninstall.

Creating Mailboxes and Folders

HERMES lets you create mailboxes to put messages in, and folders to put mailboxes (or more folders) in.

There are three ways to create mailboxes and folders:

•Using the New command from the Mailbox menu and folder submenus.

•Using the New command from the drop-down menu in the Mailboxes window (see Using the Mailboxes Window ).

•Using the New command from the Transfer menu (see Creating a Mailbox or Folder During Transfer ).

Note that depending on whether your incoming mail server uses the POP or IMAP protocol, there are some differences in the way mailboxes and mail folders are created, stored, and managed. For more information, see Mailbox and Folder Management: POP vs. IMAP Server .

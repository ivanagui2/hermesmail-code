Date

Specifies what date the message will be sent.

Change the default date if you want the message to be sent at a different date. The message is saved in the Out mailbox with a clock icon in the status column, then sent on the specified date. For the message to be sent at the correct time, Eudora must be running at that time. If Eudora is not running then, the message is sent when Eudora is next opened.

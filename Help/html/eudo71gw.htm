What's New in Eudora Email 7.0

Welcome to Eudora Email 7.0! Below describes the new features and functions added to Eudora 7.0 for Windows. More features will be added at a later release.

·Fast Indexed Search. Eudora messages are indexed for wicked fast searching.

·BossWatch (VIP Sender Warnings). Specify VIP addresses in your address book for special warnings when sending messages to avoid accidentally sending mail to special contacts.

·IMAP. Enhanced IMAP support in Eudora allows for more functions to be handled while offline.

·S/MIME. Verify signatures of S/MIME signed messages, send signed, encrypted messages..

For details on other changes in Eudora 7.0, refer to the README and RELNOTE text files that reside in the directory where Eudora 7.0 is installed.

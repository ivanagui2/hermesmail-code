Mark previewed messages as read after _ second(s)

If this is checked and the Show message preview pane option is checked, the message currently selected in the mailbox window and previewed in the preview pane is automatically marked as read (blank in the Status column) after the specified number of seconds. If this is unchecked, a previewed message will not be automatically marked as read. To return a message to the Unread status (bullet in the Status column) after it has been marked as read, press Shift+Space. Press again to return to read status.

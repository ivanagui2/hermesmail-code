Automatically Deleting Attachments

When you delete messages, you can have their attachments automatically deleted by doing the following,

1. From the Tools menu, choose Options.

2. Scroll and select Attachments. The Attachments options window appears.

3. Check the Delete attachments when emptying Trash box. Be sure the attachments are still in the Attach Directory (or the directory you have specified for attachments). If you have this option on and want to delete a message but save its attachment, move the attachment into another directory before deleting the message.

4. Click OK.

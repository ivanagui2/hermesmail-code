Attachment Type popup

The Attachment Type popup menu lets you select what format documents that you attach to outgoing messages are encoded in: Multipurpose Internet Mail Extensions (MIME), BinHex, or Uuencode.

MIME is best for recipients with MIME.
BinHex is most compatible with old Macintosh mailers and previous versions of Eudora.
Uuencode is best for communicating with older PC or UNIX systems.

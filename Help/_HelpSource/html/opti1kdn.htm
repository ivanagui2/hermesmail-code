Junk Threshold

Eudora distinguishes junk mail by a scoring system. You can adjust this scoring threshold by using the slide bar indicator. If you slide the arrow to a lower number, Eudora filters out more junk mail but may also mistakenly filter more valid mail to the Junk mailbox. If you slide the arrow to a higher number, Eudora filters less junk mail but is less likely to incorrectly filter valid mail. The default setting is 50.

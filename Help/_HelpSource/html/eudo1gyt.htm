Creating a New Database

In addition to the default directory service databases that already comes configured with Eudora, you can create new Ph, Finger, and LDAP databases for your Directory Services queries. By adding a new database, you can then query that database for names, email addresses, phone numbers, and so on. Databases are created from the Protocols section of the window, and they must be created using one of these three listed protocols.

Important:You must be able to identify the server by its description and its host name before you can add it to Directory Services.

Note:You cannot create a new database using the Eudora Address Book protocol in the list. Nickname files must be created from the Address Book window. See Using the Address Book for more details.

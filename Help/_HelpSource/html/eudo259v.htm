Dragging Messages

You can drag messages from one mailbox to another using the message summaries or the mailbox (with arrow) icon in an open message window.

·To drag a message summary to another mailbox, select the message summary (or summaries) and drag the summary into any open or minimized mailbox window, or any mailbox in the Mailboxes window. When you release the mouse button, the message is transferred.

·To drag an open message to another mailbox, hold down the mouse over the mailbox (with arrow) icon and drag the message into any open mailbox or any mailbox in the Mailboxes window. When you release the mouse button, the message is transferred.

When dragging to the Mailboxes window, pause over a closed folder and the folder temporarily opens, making its mailboxes visible so that you can complete the drop operation. The folder closes again when you complete or cancel the drop.

Note:You can only drag messages around if the Allow drag and drop transfers box is checked in the Mailboxes Options Window found in Options on the Tools menu.

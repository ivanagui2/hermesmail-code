Address(es)

The email addresses that are included in the nickname.

You can use full addresses and nicknames in this field, just be sure any nicknames you use are defined in another entry. Separate each address or nickname with a comma or a carriage return (Enter). This is the only place you can use a carriage return to separate addresses.

If you do not have a Name defined for the entry (in the Info tab), all of the addresses in this field will be included in outgoing mail.

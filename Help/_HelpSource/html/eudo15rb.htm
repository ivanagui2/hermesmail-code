Using Contextual Filing

You can highlight a word or part of the word anywhere in a message (header or text), right-click, and transfer the message to a mailbox of the same name as the word.

For whole matches, for example, if you have a mailbox called Technical, you can highlight the word "technical" anywhere in the message and transfer the entire message to the Technical mailbox.

For partial matches, for example, if you have a mailbox named Old Newsletters, you can highlight the word "Newsletter" and transfer the entire message to the Old Newsletters mailbox.

To perform a contextual filing:

1.Highlight the word.

2.Right-click the word. The contextual filing submenu appears showing the mailbox(es) with similar name(s).

3.Choose the mailbox. The message transfers to the selected mailbox.

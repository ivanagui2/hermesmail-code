Size or Age (Search messages)

If you selected Size in the criteria drop-down list, enter the number of kilobytes or click the up or down arrow counting to the kilobyte number you want to base the search on. If you click the Search button, all messages that pertain to this number, as set up in the criteria and conjunction fields, will appear in the Results window.

If you selected Age in the criteria drop-down list, enter the number of days or click the up or down arrow counting the number of aging days you want to base the search on. If you click the Search button, all messages that pertain to this number, as set up in the criteria and conjunction fields, will appear in the Results window.

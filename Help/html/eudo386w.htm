Junk Mailbox

(Paid mode only)

Once you install Eudora, Eudora asks you if you want to install a new mailbox called Junk to hold junk email messages (spam). The Junk mailbox appears in your mailbox list. The Junk mailbox is a special mailbox like the In, Out, and Trash mailboxes. It cannot be deleted or renamed. See Using the Junk Mail Filter for Incoming Messages for more information.

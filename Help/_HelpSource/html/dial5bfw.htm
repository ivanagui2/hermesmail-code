Check Mail

If this option is on, then mail checking is activated for this account in the following ways:

If you have specified a number in the Check for mail every _ minutes option in the Checking Mail Options, then when automatic mail-checks are performed at these intervals, mail for this account is checked too.

Each time you do a manual check for mail, by selecting the Check Mail command from the File menu or via another method, mail is checked for this account too.

If this option is off, then mail for this personality is not checked during manual or automatic mail-checks, except that you can override the setting and still check for the personality's mail using the Mail Transfer Options dialog box (POP account only) or the Check Mail command on the drop-down menu displayed by right-clicking in the Personalities window.

Wait for _ seconds of user inactivity before processing/filtering downloaded messages

If HERMES is retrieving mail automatically in the background, the amount of time set in this field indicates the idle time HERMES waits before delivering mail to your mailboxes and folders. For example, if you are composing a message and HERMES is retrieving mail in the background, HERMES will not deliver the mail until you stop typing for the amount of time indicated here. If you set the time for 5 seconds, HERMES delivers your mail if you stop typing for at least 5 seconds. This field is beneficial if you don’t want to be interrupted too frequently while using HERMES. If the number is large in this field, you’ll get fewer interruptions.

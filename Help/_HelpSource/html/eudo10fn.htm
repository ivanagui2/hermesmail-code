Directory Service Protocols

Protocols are servers you access in Directory Services to find information on persons by entering their name, email address, clicking The Protocols section of the Directory Services window lists the directory service protocols available to you to use when looking up individuals via these services.

Eudora comes with four installed protocols.

·Ph

·Finger

·LDAP (Lightweight Directory Access Protocol)

·Eudora Address Book

The Eudora Address Book protocol does not appear in the list until you have created an Address Book in Eudora.

Click the column heading, Registered Drivers to sort the protocols alphabetically. Shift-click the column heading to sort in reverse order.

You can resize the column by dragging the divider line to the right of the column heading to the left or right.

The use of the New Database button, shown in this section of the window, is described in Creating a New Database .

Preview Pane (Content Concentrator)

Choose one of the options from the drop-down menu for how a single message is to be concentrated in the preview pane. Compact is the default.

None—Turns off Content Concentrator. There is no change in the message format.

Compact—Message is truncated; that is, excess headers and text are removed and replaced with ellipses and a snip indicator. Only the To:, From:, Subject:, and CC: of the header fields appear.

Terse—Message is truncated further; only the From: field of the header appears in the message and the concentrated text.

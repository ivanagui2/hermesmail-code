Retrieve messages marked for retrieval

Retrieve the messages on the server that are marked Fetch or Fetch & Delete in Eudora. This is for messages that were left on the server with the Leave mail on server, Skip messages over x k, or Fetch all message headers option.

To mark a message Fetch or Fetch & Delete, either click on the icons in the message toolbar, or select an option from the Server Action drop-down menu of the message summary.

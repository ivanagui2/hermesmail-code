Remove or Add Excerpt Bars

In an outgoing message, you now have the ability to add or remove excerpt bars right in the message.

To remove excerpt bars in an outgoing message, do the following:

1.Move the cursor to the paragraph to where you want to delete an excerpt bar or select the paragraph.

2.From the Edit menu, choose Text. The Text submenu appears.

3.From the Text submenu, choose Remove Quote Level or press the Ctrl key + . (period).

One level of excerpt bar is removed.

To add excerpt bars in an outgoing message, do the following:

1.Move the cursor to the paragraph to where you want to add an excerpt bar or select the paragraph.

2.From the Edit menu, choose Text. The Text submenu appears.

3.From the Text submenu, choose Add Quote Level or press the Ctrl key + > (greater than).

One level of excerpt bar is added.

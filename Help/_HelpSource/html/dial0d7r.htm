Details List

Your query results are listed in the left half of the Directory Services window, in the results list (top view) and the details pane (bottom view). You can resize the two views relative to each other using the separator between them; drag the separator up or down.

To view the details of a result in the details pane, select the result in the results list by clicking once on it. The full data appears in the details pane. If nothing is selected in the results list, the details pane is blank. If multiple summaries are selected in the results list, the details pane lists the details for each item, with each item separated by a divider line.

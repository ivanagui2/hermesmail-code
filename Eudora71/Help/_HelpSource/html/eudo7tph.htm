Taking a Message Out of the Queue

A message that is queued but is not yet sent can be unqueued using the Change Queueing command. To take a message out of the queue, do the following:

1. Open the Out mailbox.

2. Select the desired message summary.

3. From the Message menu, choose Change Queueing and click Don’t send. This changes the message status from queued (Q) to sendable (•). The message is held in the Out mailbox until it is either deleted, re-queued, or sent. To send the message immediately, open it and click Send.

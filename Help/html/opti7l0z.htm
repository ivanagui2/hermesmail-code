Use Microsoft viewer

If this is unchecked, Eudora displays advanced formatting, graphics, and multimedia in incoming and outgoing messages using its own built-in viewer. If this is checked, Eudora takes full advantage of the Web browser capabilities of Microsoft's Internet Explorer by showing Web content right in incoming and outgoing messages or linking to the Internet, without your having to open the Web browser. This option is only available if you have Microsoft's Internet Explorer version 3.0 or higher installed and available on your system.
